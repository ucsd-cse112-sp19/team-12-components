欢迎使用 JJ Components

这是JJ 网络组件库版本1.0.0。 最新组件包括：

JJ Input Number

JJ Slider

更多组件即将推出。点击这里前往GitHub代码库。

NPM

在项目根目录使用NPM安装库。 包含所有组件。 

npm i jj-components

CDN

如果想要使用某些组件，也可以使用CDN来安装。

JJ Input Number
<script src="https://cdn.jsdelivr.net/gh/ucsd-cse112/team-12-components/packages/jj-input-number/jj-input-number.js"></script>
JJ Slider
<script src="https://cdn.jsdelivr.net/gh/ucsd-cse112/team-12-components/packages/jj-slider/jj-slider.js"></script>
