JJ Input Number

Customizable number input field with increse and decrease buttons.

Basic usage

Change range

<jj-input-number min="0" max="8"></jj-input-number>

Customized initial value

<jj-input-number min="0" max="8" value="4"></jj-input-number>

Change step

<jj-input-number min="0" max="8" step="2"></jj-input-number>

Change presition

<min="0" max="8" value="1" precision="2" step="0.5"></jj-input-number>
