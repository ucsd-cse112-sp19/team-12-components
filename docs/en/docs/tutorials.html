Tutorials

Here is a step by step tutorial with an example of JJ-Slider.

Importing the component

Find, copy and paste the CDN of the component to your html header section: 

<script src="https://cdn.jsdelivr.net/gh/ucsd-cse112/team-12-components/packages/jj-slider/jj-slider.js"></script>

Or, you could also install the whole library to your project by running the following command in terminal at your project's root directory: 

npm i jj-components

Now you have the components imported and you are ready to go!

Using your first component

Now, let's get the component going and display it in your web page or web app!

Generally you use web components just like any other html tags. For example in this case you could use the JJ-Slider with the following html tags: 

<jj-slider></jj-slider>

You could place the component to any place you want! After you place the tags in your html page, you could open the page up and check the component out! You should see JJ-Slider with all default attributes just like the following:

You just successfully used your first JJ Component for the first time!

Change Attributes

Now, let's try to change the attributes of the component and customize it for your need.

You could check our what are the attributes in their documentation pages. In this case the documentation for the JJ Slider is here

Let's say you are selling t-shirts. One customer could only purchase 8 of them at a time. You want to set the default to 4 t-shirts, right in the middle, so customers would not need to move their cursor to far every time.

So the slider should have the following properties:

1. Range from 0 to 8

2. Initial value is 4

JJ Slider gets you covered with the following attributes:

1. min and max for the range

2. value for the initial value

You could achieve that by writing out the attributes like this:

<jj-slider min="0" max="8" value="4"></jj-slider>

And you get the following slider you want!
