How to Install

Using NPM

Install Jay Jay Components Library with NPM. 

npm i jj-components

Using CDN

Import any component in Jay Jay Components Library into your library with the CDNs. 

<script src="https://cdn.jsdelivr.net/gh/ucsd-cse112/team-12-components/packages/jj-slider/jj-slider.js"></script>
